!! This mod requires LATEST version of Geckolib.

Starless is a mod that adds some entities for player to interact with. They can be friendly, neutral or hostile, depending on entity and type of day they spawn at.

Download released version here: https://github.com/RedPa1m/Starless/releases/tag/playtest
